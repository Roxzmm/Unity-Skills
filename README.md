# Unity-Skills

A collection of Claude Code skills for Unity game development.

## Plugins

| Plugin | Description | Install |
|---|---|---|
| [unity-render-event-analysis](plugins/unity-render-event-analysis/) | Analyze per-frame draw-call data — shaders, passes, keywords via FrameDebugger reflection | `/plugin install unity-render-event-analysis@Unity-Skills` |

## Installation

```bash
# Add this marketplace
/plugin marketplace add Roxzmm/Unity-Skills

# Install a plugin
/plugin install unity-render-event-analysis@Unity-Skills
```

## Usage

After installing a plugin, invoke its skill(s) by describing your task naturally.
The skill will be loaded automatically when relevant.
