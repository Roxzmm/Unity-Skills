# Unity-Skills

A collection of Claude Code skills for Unity game development.

Unity 游戏开发 Claude Code 技能合集。

> **Language Policy** — All skills/plugins default to English with Chinese (中文) descriptions included.  
> **语言政策** — 所有技能/插件默认英文，同时附带中文说明。

## Plugins

| Plugin | Description | 说明 | Install |
|---|---|---|---|
| [unity-render-event-analysis](plugins/unity-render-event-analysis/) | Analyze per-frame draw-call data — shaders, passes, keywords via FrameDebugger reflection | 通过 FrameDebugger 反射逐帧分析渲染数据 — 着色器、Pass、关键字 | `/plugin install unity-render-event-analysis@Unity-Skills` |

## Installation / 安装

```bash
# Add this marketplace / 添加市场
/plugin marketplace add Roxzmm/Unity-Skills

# Install a plugin / 安装插件
/plugin install unity-render-event-analysis@Unity-Skills
```

## Usage / 使用

After installing a plugin, invoke its skill(s) by describing your task naturally.
The skill will be loaded automatically when relevant.

安装插件后，通过自然描述任务来调用技能，相关技能会自动加载。
