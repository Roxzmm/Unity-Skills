# Unity-Skills

Unity 游戏开发的 Claude Code 技能合集。

## 插件列表

| 插件 | 说明 | 安装 |
|---|---|---|
| [unity-render-event-analysis](Plugins/unity-render-event-analysis/) | 通过 FrameDebugger 反射逐帧分析渲染数据 — 着色器、Pass、关键字 | `/plugin install unity-render-event-analysis@Unity-Skills` |

## 安装方法

```bash
# 添加市场
/plugin marketplace add Roxzmm/Unity-Skills

# 安装插件
/plugin install unity-render-event-analysis@Unity-Skills
```

## 使用方法

安装插件后，通过自然描述任务来调用技能。当描述与技能相关时，技能将自动加载。
